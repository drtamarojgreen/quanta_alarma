# QuantaAlarma Glossary

- **Finding**: A specific instance where an analysis rule matched the input text.
- **Rule**: A predefined pattern (regex) and associated metadata used for detection.
- **Risk Score**: A normalized value (0-100) representing the overall risk of the input text.
- **Policy**: A configuration file defining rules, severity weights, and failure thresholds.
- **Suppression**: Marking a finding as known or acceptable, excluding it from the risk score.
- **Fingerprint**: A unique identifier for a finding, used for stable tracking and suppressions.
